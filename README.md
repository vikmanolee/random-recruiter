# The Random Recruiter

## Live preview

Also available on vikmanolee.github.io/random-recruiter .

## How to use

Download the app

```sh
cd random-recruiter
```

Install it and run:

```sh
npm install @material-ui/core
npm start
```

## The idea behind the example
